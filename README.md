# Arduino Uno R4 WiFi - E-Paper "Hello World"

Αυτό το έργο είναι ένα απλό παράδειγμα (sketch) για το **Arduino Uno R4 WiFi**, το οποίο επιδεικνύει πώς να οδηγήσετε μια οθόνη **E-Paper (E-Ink)**. Ο κώδικας αρχικοποιεί την οθόνη και εμφανίζει το μήνυμα "Hello World".

## 📝 Περιγραφή Λειτουργίας

Ο κώδικας εκτελεί τις εξής ενέργειες μόλις τροφοδοτηθεί το Arduino (`setup`):
1.  **Αρχικοποίηση:** Ξεκινά τη σειριακή επικοινωνία και ενεργοποιεί την οθόνη E-Paper.
2.  **Ρύθμιση:** Ορίζει τον προσανατολισμό (Rotation) και το παράθυρο ενημέρωσης.
3.  **Εκτύπωση:**
    * Καθαρίζει την οθόνη με λευκό φόντο.
    * Γράφει τη λέξη **"Hello"** με μαύρο χρώμα.
    * Γράφει τη λέξη **"World"** (ο κώδικας περιέχει εντολή για κόκκινο χρώμα, αλλά εξαρτάται από τον τύπο της οθόνης που έχει οριστεί).
4.  **Αδράνεια:** Στο `loop` δεν εκτελείται καμία ενέργεια, καθώς η εικόνα στο E-Paper παραμένει σταθερή χωρίς κατανάλωση ενέργειας.

## 🛠️ Απαιτούμενος Εξοπλισμός

* **Arduino Uno R4 WiFi**
* **Οθόνη E-Paper** (Συμβατή με τη βιβλιοθήκη `GxEPD2`, π.χ. 1.54").

## 🔌 Συνδεσμολογία (Wiring)

Ο κώδικας χρησιμοποιεί συγκεκριμένα Pins για το Arduino Uno R4 WiFi:

| Pin Οθόνης | Pin Arduino | Ορισμός στον Κώδικα |
| :--- | :--- | :--- |
| **CS** | **10** | `static const uint8_t EPD_CS = 10;` |
| **DC** | **9** | `static const uint8_t EPD_DC = 9;` |
| **RST** | **8** | `static const uint8_t EPD_RST = 8;` |
| **BUSY** | **7** | `static const uint8_t EPD_BUSY = 7;` |
| **SCK** | **13** | (Default SPI CLK) |
| **MOSI** | **11** | (Default SPI MOSI) |

> **Σημαντικό:** Βεβαιωθείτε ότι η οθόνη τροφοδοτείται με **3.3V** και όχι 5V.

## 📚 Βιβλιοθήκες

Για να κάνετε compile, χρειάζεστε τις εξής βιβλιοθήκες στο Arduino IDE:
* `GxEPD2` (Διαχείριση E-Paper)
* `Adafruit GFX` (Γραφικά)
* `SPI` (Επικοινωνία)

## ⚙️ Ρυθμίσεις Κώδικα

Στον κώδικα υπάρχουν σχόλια για διαφορετικούς τύπους οθονών:
* **Τρέχουσα Ρύθμιση:** Είναι ενεργοποιημένη η κλάση `GxEPD2_BW` για ασπρόμαυρη οθόνη 1.54" (μοντέλο `GxEPD2_154_D67`).
* **Εναλλακτική:** Υπάρχουν σχολιασμένες γραμμές για τρίχρωμες οθόνες (Μαύρο/Άσπρο/Κόκκινο) μέσω της κλάσης `GxEPD2_3C`.

Αν η οθόνη σας είναι διαφορετική, θα πρέπει να απο-σχολιάσετε (uncomment) την κατάλληλη γραμμή ορισμού του αντικειμένου `display`.

## 🚀 Εγκατάσταση

1.  Συνδέστε την οθόνη στο Arduino σύμφωνα με τον πίνακα συνδεσμολογίας.
2.  Ανοίξτε το `main.cpp` με το Arduino IDE.
3.  Επιλέξτε το σωστό board (Arduino Uno R4 WiFi) και Port.
4.  Πατήστε **Upload**.
